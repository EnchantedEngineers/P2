# Enchanted Store aka E-Store

## Overview

### This is an e-commerce site for users to buy our products. 

<br>

#### Products
- Vegetables/Fruits
- Meat
- Snacks

#### Functionality
- Login/registration
- Show user profile
- Display products
- Add products on the cart
- Checkout products from cart/remove the product
- View purchase history
- Return a product
- Adding new product
